Opdracht 1

Onderstaand HTML fragment maakt deel uit van een menubalk op een bedrijfswebsite:

<ul>
    <li class="menuitem"><a href="...">products</a></li>
    <li class="menuitem"><a href="...">distributors</a></li>
    <li class="menuitem"><a href="...">contact</a></li>
</ul>

De items in de menubalk worden gestyled met:

.menuitem {
    display: inline-block;
    width: 150px;
}
.menuitem > a {
    color: red;
}

Herschrijf dit fragment en de CSS-regels zonder classes op elk <li> element te moeten gebruiken.

Oplossing

De herschreven versie van het HTML-fragment en de CSS-regels zonder gebruik van de menuitem class op elk <li> element:

<ul>
    <li><a href="...">Products</a></li>
    <li><a href="...">Distributors</a></li>
    <li><a href="...">Contact</a></li>
</ul>

De CSS-regels worden als volgt aangepast:

ul > li {
    display: inline-block;
    width: 150px;
}
ul > li > a {
    color: red;
}

Uitleg van de CSS-selectors:

  • ul > li: Selecteert alle <li> elementen die directe kinderen zijn van <ul>, wat inline-styling en breedte toepast.
  • ul > li > a: Selecteert de <a> tags binnen de <li> elementen, waardoor de linktekst rood wordt.
