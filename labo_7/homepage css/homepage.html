Vraag 3

De CSS-regel .onbekend {background-color: red;} werkt niet op een cel in een <tr> met de class `alt`. Dit komt door specifiteit: .vrienden table tbody .alt td heeft een score van 23 punten (10 voor elke class + 3 voor elementen), terwijl .onbekend slechts 10 punten heeft. Een oplossing is: .vrienden table tbody .alt td.onbekend {background-color: red;}
