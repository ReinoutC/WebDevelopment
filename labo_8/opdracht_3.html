Opdracht 3

Waarom staat de tekst in de blokken horizontaal gecentreerd?

  • De tekst in de blokken wordt horizontaal gecentreerd door de toepassing van text-align: center; in de CSS voor de elementen. Deze property zorgt ervoor dat de tekst in het midden van het element wordt geplaatst, ongeacht de breedte van het element.

Ga na dat een property vertical-align: center; geen effect heeft. Waarom eigenlijk niet?

  • De property vertical-align: center; bestaat niet in CSS. De vertical-align property kan alleen bepaalde waarden aannemen zoals top, middle, bottom, enz., maar er is geen center waarde.

Ga na dat vertical-align: middle; er niet voor zorgt dat de tekst verticaal gecentreerd wordt. Wat doet vertical-align dan wel?

  • De vertical-align property wordt gebruikt voor inline-elementen of elementen die inline-block zijn en werkt in de context van hun omgeving. Het zorgt ervoor dat een element uitgelijnd wordt ten opzichte van het omliggende element. In dit geval heeft vertical-align: middle; geen effect op de tekst omdat het geen invloed heeft op de plaatsing van de inhoud binnen een block-level element.

Wat is de ‘simpele’ manier om een enkele regel tekst verticaal te centreren? (Tip: line-height)

  • De eenvoudige manier om een enkele regel tekst verticaal te centreren in een element is door de line-height gelijk te maken aan de hoogte van het element. Hierdoor zal de enkele regel tekst netjes in het midden staan.

Welke truc wordt vaak gebruikt om meerdere regels verticaal te centreren? (Tip: table cell)

  • Een veelgebruikte truc om meerdere regels tekst verticaal te centreren, is door het element te stylen alsof het een tabelcel is, bijvoorbeeld met display: table-cell; en vertical-align: middle;. Dit zorgt ervoor dat de tekst netjes verticaal wordt uitgelijnd.

In 'styles.css' staan twee verschillende CSS regels die de height instellen van een element met class="area tall". Waarom krijgt de regel voor class 'tall' voorrang?

  • De regel die specifiek gericht is op de class tall krijgt voorrang omdat deze specifieker is dan een algemene selector. In CSS krijgt een meer specifieke selector (zoals een class-selectie) voorrang boven een algemene regel. Bovendien kan de plaats van de regel in de CSS-file en het gebruik van !important invloed hebben op welke regel voorrang krijgt.

Verander de CSS-file om je vermoeden te bevestigen. Zet nadien alles weer terug op z’n oorspronkelijke plaats.

  • Door de CSS-regels in de stylesheet aan te passen, bijvoorbeeld door de volgorde van de regels te veranderen, kun je testen welke regel prioriteit krijgt. De regel die later in de stylesheet staat (met dezelfde specificiteit), zal voorrang krijgen. Na deze wijziging kun je bevestigen dat de volgorde van de regels invloed heeft op welke CSS-regel toegepast wordt.
